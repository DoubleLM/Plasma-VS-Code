01 | New CSS Feature

CSS caret-color

The caret-color property allows the color to be set of the caret (blinking cursor) in an input field. This property is useful for setting the color of the caret to match the text color of the input field. The caret-color property is supported in all major browsers except Internet Explorer.

I like this property because not only is it objectively funny to mess with, it also allows for more variety with background design and can be used to better contrast against "dark mode" backgrounds. It's a small detail that can make a big difference in the overall look of a website.

02 | My Favourite Learned CSS Feature

My favourite CSS feature that I've learned so far has to be the flexible box model. It's a simple and intuitive way to lay out, align, and distribute space among items in a container, even when their size is unknown or dynamic. It's a great way to create complex layouts with minimal code and is supported in all major browsers.

Furthermore, I've also learned how to make it responsive in its design, which is a great way to make sure that the layout looks good on all sorts of devices and screen sizes. Check it out for yourself by resizing this window and paying attention to the Flags I Like section! It'll transform from four columns down to two. This is thanks to code I learned on W3Schools.

03 | AI and Web Development

Artificial Intelligence makes for a dangerous companion to have in the field of web development, at least during its current infancy. Its lack of abject reliability is a major drawback, but its potential for quickly filling out the minutiae for you is a major boon. It's a double-edged sword that developers will have to decide for themselves whether they would like to play with or not.
